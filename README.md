  ASTU Lost & Found

ASTU Lost and Found is a web application designed to help students at Adama Science and Technology University (ASTU) locate lost items and return found items efficiently.

🚀 Features
##User Authentication: Secure login and registration for ASTU community members.

##Item Posting: Users can post details, photos, and locations of lost or found items.

##Search & Filter: Easily find items by category, keyword, or date.

##Claim System: Integrated system to claim items, with admin approval functionality.

##Real-time Notifications: Get notified when an item is found or a claim is approved.

 Technology Stack

 Frontend
     React.js
     Axios for API requests
     React Toastify for notifications
     React Router for navigation

  Backend
     Node.js
     Express.js
     MongoDB (or database of choice)

📋 Installation & Setup Prerequisites

Node.js installed
MongoDB installed and running

###Steps
    Clone the repository

  Bash
   git clone https://github.com/ephraimtessema-lgtm/astu_lost_and_found.git
   cd astu_lost_and_found
   Install Backend Dependencies

  Bash
   cd backend
   npm install
   Install Frontend Dependencies

  Bash
    cd ../frontend
    npm install
    Configure Environment Variables

##Create a .env file in the backend folder and add your database URI and port.

##Create a .env file in the frontend folder if needed (e.g., for API URLs).

##Run the Application

   Backend: cd backend && npm start
   Frontend: cd frontend && npm start
